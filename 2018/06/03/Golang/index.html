初识go语言

Golang

Go是一种开源的程序设计语言，它意在使得人们能够方便的构建简单、可靠、搞笑的软件——golang.org

【注意】：

  1. main函数实现的就是入口功能。
  2. go语言中的函数前面必须加上func这个单词，而且函数名称后面必须跟一对小括号
  3. go语言中区分大小写。Print函数P大写
    【注意2】：
  4. 强制左花括号 { 的放置位置，如果把花括号{ 另起一行，go不允许，会报错
  5. 每行代码占一行。
  6. Print、Println（换行）、Printf（%d、%s、%t、%T等格式化输出）
  7. %t匹配bool类型返回值true和false
  8. %T匹配其变量对应的数据类型

变量

变量声明、赋值、初始化

自动推导类型（更简单的变量赋值方式）

num：=30

go中：自动退到类型就是不通过var声明变量，不指定类型直接在变量名后面跟：号，同时完成赋值，go会根据所赋的值自动推导出变量的类型

多重赋值、匿名变量

  1. 多重赋值：a,b,c:=1,2.5,”abcddf”
  2. 匿名变量：匿名变量，丢弃数据不进行处理，\匿名变量配合函数返回值使用才有价值
    语法：, i,\,j:=1,2,3,4

fmt包的格式化输出输入

格式 含义
%% 一个%
%d 一个二进制整数值（基数为2），或者是一个（高级的）用科学记数法表示的指数为2的浮点数
%c 字符型。可以把输入的数字按照ASCII码相应转换为对应的字符
%d 一个十进制数值（基数为10）
%e 以科学计数法e表示的浮点数或者复数值
%E 以科学计数法e表示的浮点数或者复数值
%f 以标准计数法表示的浮点数或者复数值
%g 以%e或者%f表示的浮点数或者复数，任何一个都以最为紧凑的方式输出
%G 以%E或者%f表示的浮点数或者复数，任何一个都以最为紧凑的方式输出
%o 一个以八进制表示的数字（基数为8）
%p 以十六进制表示的一个值的地址，前缀为0x，字母使用小写的a-f表示
%q 使用go语法以及必须时使用转义，以双引号括起来的字符串或者字节切片[]byte,或者是以单引号括起来的数字
%s 字符串。输出字符串中的字符直至字符串中的空字符（字符串以‘\0’结尾，这个’\0’即空字符）
%t 以true或者false输出的布尔值
%T 使用go语法输出的值的数据类型
%U 一个用Unicode表示法表示的整型码点，默认值为4个数字字符
%v 使用默认格式输出的内置或者自定义类型的值，或者是使用其类型的String（）方式输出的自定义值，如果该方法存在的话
%x 以十六进制表示的整型值（基数为十六），数字a-f是用小写表示
%X 以十六进制表示的整型值（基数为十六），数字A-F是用小写表示

算术运算符、 赋值运算符、关系运算符、逻辑运算符

  1. 算术运算符
运算符 术语 示例 结果
+ 加 10+5 15

++ 后自增（go中没有前自增） a=0;a++ a=1（a++等价a=a+1）

【go语言中没有前缀自增自减】

  • go语言中不允许隐式转换，所有类型转换必须显式声明（强制转换），而且转换只能发生在两种相互兼容的类型之间

  • 强制类型转换写法：数据类型名（待转换的值）

  • 【注意】：
  • int强转float 多小数
  • float强转int丢精度

  1. 赋值运算符
运算符 说明 示例
= 普通赋值 c=a+b将表达式结果赋值给c
+= 相加后再赋值 c+=a等价 c=c-a
-= 相减后再赋值 c-=a等价于 c=c-a
*= 相乘后再赋值 c=a等价于 c=ca
/= 相除后再赋值 c/=a等价于 c=c/a
%= 求余后再赋值 c%=a等价于 c=c%a
  1. 关系运算符（比较运算符）
运算符 术语 示例 结果
== 相等于 4==3 false
！= 不等于 4！=3 true
< 小于 4<3 false
> 大于 4>3 true
<= 小于等于 4<=3 false
>= 大于等于 4>=1 true
  1. 逻辑运算符
运算符 术语 示例 结果
! 非 ！a 如果a为假，则！a为真；如果a为真，则！a为假
&& 与 a&&b 如果a和b都为真，则结果为真，否则为假。
|| 或 a||b 如果a和b有一个为真，则结果为真，二者都为假时，结果为假
  1. 其他运算符
