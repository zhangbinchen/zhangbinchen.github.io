函数

概念

  • 完成某一特定功能的代码段
  • 封装函数的目的，代码的复用、模块化项目文件结构
  • 程序就像公司，公司由部门组成，部门类似于函数

函数的分类

  • 系统函数(库函数)：

    printf、scanf、puts、gets、。。。。。。。。。。
    掌握其函数功能、函数原型：函数名、函数返回值。
    调用：引入对应头文件

  • 用户自定义函数

    函数的定义

  • 函数定义的一般形式
    返回类型 函数名（形式参数列表）
    {
    数据定义部分
    执行语句部分
    }
  • 函数的名字、形参、函数体、返回值
    • 函数名：
      代表这个为函数，不是普通的变量名—–见名知意
    • 形参列表
      在未出现函数调用时，他们并不占内存的存储单元，形参里的变量不能赋值
      在定义函数时指定的形参，必须是：类型+变量 的形式
      定义函数时如果没有形参，圆括号内容为空，或写一个void关键字
      形参出现在函数定义中，在整个函数体内可以使用，离开该函数则不能使用
    • 函数体
      函数有返回值，必须写return关键字。返回对应类型的数据。
      函数无返回值，可以不写。也可以写：return；
    • 函数的声明：
      c语言中，有0-N次
      向编译器注册该函数。
      包含:函数原型+分号结束标记
      函数原型（函数接口）：返回值类型、函数名、形参列表；
      void my_print(char str[]);
      int add(“int a,int b”);
      使用场景：在函数调用之前，没有函数定义时，需要使用函数声明
    • 函数调用：
      c语言中，有0-N次
      使用该函数，实现对应的功能
      语法：函数名（实参列表）；
      实参列表：与形参类型严格匹配（类型、个数、顺序）的变量或常量。不带类型名。如果形参列表为void，那么省略实参，不可以写void
      实参出现在主调函数中，进入被调函数后，实参也不能使用
      实参变量对形参变量的数据传递是：“值传递”，只由实参传给形参，而不能由形参传回来给实参
      在执行一个被调函数时，形参的值如果发生改变，并不会改变主调函数中实参的值
    • 无参函数的调用
      如果是调用无参函数，则不能加上“实参”，但括号不能省略
  • exit（）函数

    头文件:#include <stdlib.h>
    功能：退出当前程序
    return:返回当前函数调用，返回给调用者
    在main（）函数中调用return和exit结果一样，

  • exit main
  • 获取随机数(函数调用)

多文件联编：

  • 向工程中添加多个.c，但只有一个.c中包含main函数，其他的.c中都是功能函数
  • 对应给每一个模块文件(.c)添加函数声明到 xxx.h中
  • 声明语法：int add（int a,int b）;

    防止头文件被重复包含：

  1. #pragma once

  2. #ifndef HEAD_H
    #define HEAD_H
    ….(函数声明语句)
    #endif
